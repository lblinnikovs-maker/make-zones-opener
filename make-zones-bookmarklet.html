MakeAdmin All

Opens the current Make admin or native app URL across all production zones in new tabs.

MakeAdmin All
  1. Allow popups: In Chrome, go to Settings → Privacy and security → Site settings → Pop-ups and redirects, then add file://* (or your make.com domain) to the Allowed list.
  2. Drag the purple button above to your bookmarks bar.
  3. Navigate to any *.make.com URL in the browser.
  4. Click the bookmark — all zones open in new tabs.
